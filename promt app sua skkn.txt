### **BƯỚC 3 : ĐIỀU CHỈNH CÁC PHẦN THEO TÊN ĐỀ TÀI MỚI**

Sau khi người dùng chọn tên đề tài mới (giả sử chọn #1), AI sẽ tự động đề xuất sửa từng phần.

#### **Nguyên tắc Sửa chữa (QUAN TRỌNG) :**

```plaintext
🔒 NGUYÊN TẮC SỬA CHỮA - KHÔNG VI PHẠM

✅ ĐƯỢC PHÉP SỬA:
   1. Tên đề tài, tiêu đề các phần
   2. Câu văn chung chung, sáo rỗng, dễ trùng lặp
   3. Cách diễn đạt, ngôn ngữ (paraphrase)
   4. Bổ sung ví dụ cụ thể, số liệu giả định hợp lý
   5. Cấu trúc câu, đoạn văn
   6. Thuật ngữ chuyên môn (nâng cấp)

❌ TUYỆT ĐỐI KHÔNG SỬA:
   1. Số liệu thực nghiệm của giáo viên (điểm số, tỷ lệ, thống kê)
   2. Tên trường, lớp, học sinh, địa điểm cụ thể
   3. Thời gian thực hiện (ngày/tháng/năm)
   4. Trải nghiệm cá nhân, quan sát thực tế của giáo viên
   5. Kết quả đã đạt được (giải thưởng, phản hồi, etc.)
   6. Phương pháp nghiên cứu đã thực hiện

🎯 MỤC TIÊU: Giữ nguyên "xương sống" (dữ liệu thực), thay đổi "da thịt" (cách diễn đạt)
```

```python
def check_plagiarism(text, threshold=0.8):
    # Bước 1: Chia nhỏ văn bản thành đoạn (chunk)
    chunks = split_into_chunks(text, size=100)
    
    # Bước 2: Kiểm tra từng chunk
    results = []
    for chunk in chunks:
        # Layer 1: Check internal database
        db_match = check_database(chunk)
        
        # Layer 2: Google Search
        if db_match < threshold:
            search_match = google_search(chunk)
        
        # Layer 3: Specific websites
        if search_match < threshold:
            web_match = scrape_edu_websites(chunk)
        
        similarity = max(db_match, search_match, web_match)
        if similarity > threshold:
            results.append({
                'chunk': chunk,
                'similarity': similarity,
                'source': find_source(chunk)
            })
    
    return results
```

```python
def generate_new_titles(old_title, content, n=5):
    # Bước 1: Trích xuất từ khóa
    keywords = extract_keywords(old_title, content)
    
    # Bước 2: Phân tích nội dung để tìm điểm độc đáo
    unique_points = analyze_unique_aspects(content)
    
    # Bước 3: Áp dụng 10 công thức đặt tên
    formulas = [
        "{tool} + {method} + {specific_object}",
        "{product} + {capability_goal}",
        "{new_tech} + {real_problem}",
        # ... 7 công thức khác
    ]
    
    candidates = []
    for formula in formulas:
        title = apply_formula(formula, keywords, unique_points)
        score = evaluate_title(title)  # Tính điểm độc đáo
        candidates.append((title, score))
    
    # Bước 4: Kiểm tra trùng lặp
    candidates = [(t, s) for t, s in candidates 
                  if check_title_uniqueness(t) > 0.9]
    
    # Bước 5: Xếp hạng và trả về top N
    return sorted(candidates, key=lambda x: x[1], reverse=True)[:n]
```

```python
def paraphrase_paragraph(text, preserve_data):
    # Bước 1: Nhận diện dữ liệu cần giữ nguyên
    protected = identify_protected_data(text, preserve_data)
    # VD: số liệu, tên riêng, ngày tháng
    
    # Bước 2: Tách phần cần sửa
    editable = extract_editable_parts(text, protected)
    
    # Bước 3: Paraphrase từng câu
    new_sentences = []
    for sentence in editable:
        # Sử dụng AI model
        paraphrased = ai_paraphrase(
            sentence,
            style="academic",
            preserve=protected
        )
        new_sentences.append(paraphrased)
    
    # Bước 4: Ghép lại, giữ nguyên dữ liệu bảo vệ
    return reconstruct_paragraph(new_sentences, protected)
```
